# 🌿 CareConnect – Mini Healthcare Support App

CareConnect is a simple web application designed to help NGOs connect patients, volunteers, and healthcare providers in one place.

---

## 🏥 What It Does

- 🧑‍⚕️ Allows patients to request healthcare support
- 🤝 Enables volunteers to register and help
- 🏥 Connects clinics and partners
- 💬 Provides instant guidance using an AI chatbot

---

## 🤖 AI Feature
